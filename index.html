学习资料

书籍：《编程小白的第一本 python 入门书》

Python版本：python 3 （官网上下载最新的版本即可，截止到目前最新版是 3.6）

python3对初学者更友好，因为python2对unicode字符的处理会让初学者疑惑； 另外，python3的语法特性更加现代化，更好的面向对象支持，python3是python的未来发展方向

网页版教程：廖雪峰的《Python教程》

视频教程：廖雪峰的《Pytghon入门》

注意：视频教程是基于python2.7讲解的，在语法上有一些不兼容的地方，如果遇到问题，请在微信群里及时说出来。

进度01

学习时间：2天

学习内容：《编程小白的第一本 python 入门书》

  • 第一章：为什么选择Python
  • 第二章：现在就开始

学习目的：

  1. 对python有一个大致了解，Python能做什么，主要的应用场景。
  2. 部署Python开发环境，这是学习python的第一个步骤。可以按照书中所写试用重量级IDE工具PyCharm，或者使用Sublime。建议选择PyCharm。
  3. Hello World。

练习：

  1. 写出你学习python的原因及目的
  2. 在控制台或者IDE中，写出第一个程序：打印“Hello World”

进度02

学习时间：1个月

学习内容：《编程小白的第一本 python 入门书》

  • 第三章：变量与字符串
  • 第四章：函数的魔法
  • 第五章：循环与判断
  • 第六章：数据结构

学习目的：

  1. 学习结构化编程（通用知识，几乎所有当下流行的编程语言都差不多）
  2. 交流，交流，交流，重要的事情说三遍。在进行进度02的时候一定会有很多疑问，一定要和大家多交流，把疑惑说出来，也许你的疑惑他人也遇到过并且花了大量时间和力气搞明白了，如果你觉得时间比较珍贵，那就多交流，少走弯路

练习：

  1. 把工作中，你认为可以自动化的事情列出来，尽量详细

  2. 大量练习，见“46 Simple Python Exercises”

    注意：这些练习中，肯定会有超出《编程小白的第一本 python 入门书》这本书中的内容，所以要查找资料自学。
